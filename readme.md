# Grocery Website

## Overview

This project is a responsive website built using HTML, CSS, and various libraries. The website showcases a grocery store with different sections such as services, popular products, customer reviews, and special offers. The website is designed to be visually appealing and user-friendly.

## Features

### Navigation Bar
- **Logo**: Displays the store's logo.
- **Nav Links**: Links to different sections of the website (Product, Services, Contact us).
- **Icons & Buttons**: Includes search and cart icons, and buttons for Login and Register.
- **Hamburger Menu**: A dropdown menu for mobile view.

### Hero Section
- **Heading**: A large heading with an animation effect.
- **Paragraph**: A brief description of the store's offerings.
- **Image**: An image of fresh vegetables.

### Services Section
- **Service Cards**: Three cards highlighting the store's services:
  - 24/7 Services
  - Fast Delivery
  - Healthy Products

### Popular Products Section
- **Main Card**: A card with a special offer.
- **Products Grid**: A grid displaying various popular products with ratings and prices.

### Customer Reviews Section
- **Review Cards**: Three cards with customer reviews and images.

### Arrival & Offers Section
- **Offer Cards**: Two cards showcasing special offers on products.

### Footer
- **Grocery News Card**: A subscription form for grocery news.
- **Footer Links**: Links to different sections of the website.
- **Footer Socials**: Social media icons for following the store.

## Technologies Used

- **HTML**: The structure of the website.
- **CSS**: Custom styles for the website.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **DaisyUI**: A Tailwind CSS component library.
- **Font Awesome**: Icons for social media and other elements.
- **Google Fonts**: Custom fonts (Sora and Poppins) for typography.
